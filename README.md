# Microbiome study of Atopic Dermatitis in South African chidlren

*(Chen et al., submitted to BMC Microbiology, 2025)*

---

This repository contains the analysis code, processed data, and figure scripts for the manuscript:  
> **“Environmental and skin–nasal microbiome variation in South African children with atopic dermatitis.”**

The study examines how **urban vs. rural environments**, **disease severity**, and **inter-site bacterial overlap** are associated with the **skin and nasal microbiomes** of South African toddlers (ages 9–37 months) with atopic dermatitis (AD) and healthy skin controls.

## Data availability
16S rRNA gene sequencing data are deposited in the NCBI Sequence Read Archive (SRA) under Bioproject ID [PRJNA1269634](https://www.ncbi.nlm.nih.gov/bioproject/?term=PRJNA1269634).  
Data processing can be found on QIITA via study ID [15564](https://qiita.ucsd.edu/study/description/15564).
